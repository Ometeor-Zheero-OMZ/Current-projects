概要：
こちらのアプリケーションではユーザーの人間関係をデータベースに登録し、
SQLiteにて追加したデータを閲覧することができる。UIは誰にでも使えるように簡素化した。

機能：
人物の名前、ふりがな、性別、生年月日、どういった関係か、関係の度合い、追加した日をユーザーの入力で
データを追加することができる。データを追加後、friend_management.dbに保存される。次にDB Browser(SQLite)でデータベースを開き、
テーブルで管理することができる。


記入例
名前：
田中弘

ふりがな：
たなかひろし

性別：
男性

生年月日：
1998年4月22日

人間関係：
友人

関係度合い：
80点

記録日：
自動で補完されます。


使用した技術一覧
Python 3.10.4
Tkinter
Tkinter -> messagebox
datetime
sqlite3
