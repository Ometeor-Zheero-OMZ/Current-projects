概要：
ユーザーが他のサービスを利用する際にパスワードを新しく決めなければいけない工程を円滑にするために、
こちらのシステムを作った。

機能：
ユーザーは(1)サービス名(2)ご希望の長さ(3)アルファベット(4)数値(5)特殊文字を入力する。
ご希望の長さはパスワードの長さを決め、アルファベット、数値、特殊文字の数をそれぞれ決めることができる。
もし、その三点の合計の長さがご希望の長さのパスワードよりも長くなる場合はポップアップで忠告が表示される。
一方、それらがご希望の長さのパスワードよりも短い場合は、足りない分だけ文字を自動で追加する。
ご利用中のサービス名と生成されたパスワードはCSVファイルにて保存されるので、CSVファイルからコピペが容易にできる。


使用した技術一覧
